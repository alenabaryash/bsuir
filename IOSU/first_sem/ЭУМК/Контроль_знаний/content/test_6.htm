Функции:

Определение функций:

 

-         выполняет запрос к БД, определяемой структурой db. Применяется для любых запросов кроме SELECT. Возвращает код выполнения запроса (0 – удачно, либо код ошибки);

-         установливает соединение с БД, возвращает указатель на структуру db, описывающую характеристики этого соединения;

-         выполнить запрос на извлечение данных. Возвращает структуру result, содержащую результаты выполнения запроса;

-         закрыть соединение с базой данных;

-         извлечь следующую запись из структуры result;
